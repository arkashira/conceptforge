# ConceptForge

The new **ConceptForge** feature allows users to generate high‑level product concepts directly from natural language prompts. It integrates with the existing AI‑assisted workflow, providing:

- **Prompt‑based concept generation** – Input a brief description and receive a structured concept outline.
- **Design skeletons** – Automatic generation of UI wireframes and component hierarchies.
- **Export options** – JSON, Markdown, or SVG export for quick sharing.

> **Getting Started**
> 
> 1. Install the latest package: `pip install axentx-conceptforge`
> 2. Run the CLI: `conceptforge generate "New e‑commerce checkout flow"`
> 3. Review the output and refine with the interactive editor.

For detailed usage, see the [ConceptForge documentation](docs/conceptforge.md).
